# Sushi Shop

A modern frontend project for a sushi delivery store built with **React**, **TypeScript**, and **Vite**.

## Technologies Used

- React
- TypeScript
- Vite
- React Router
- ESLint

## Getting Started

```bash
npm install
npm run dev

## Project Structure

src/pages — Pages like Home, Login, Cart, etc.

src/components — Reusable components like Header.

public/ — Static assets.
